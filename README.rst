Sudoku problem reduction to SAT solver
======================================

Forked from simple-sat `simple-sat <https://github.com/sahands/simple-sat>`_.
Sudoku reduction to SAT problem is added.


Example Usage
-------------

.. code-block:: text

    $ python sudoku.py
